# Apresentação

Projeto simples de apresentação pessoal em HTML, CSS e JavaScript.

## Sobre
Este site mostra informações do autor, foto, dados de contato e uma seção "Sobre mim".

## Tecnologias
- HTML
- CSS
- JavaScript

## Como usar
Acesse diretamente por https://vinicurio.github.io/apresentacao/
ou
1. Abra `index.html` no navegador.
2. Edite os textos e a imagem conforme desejar.

## Objetivo
Criar uma página de apresentação simples e resumida para compartilhar com recrutadores ou redes sociais.
